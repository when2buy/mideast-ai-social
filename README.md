# 中东AI虚拟角色社交平台

Middle East AI Virtual Character Social Platform

## 项目概述

面向中东市场（阿拉伯语用户）的AI虚拟角色社交平台，对标 Talkie 和 Linky AI。

## 项目结构

```
├── docs/           # 需求文档、研究报告、设计稿
├── frontend/       # iOS 前端（React Native / Flutter）
├── backend/        # 后端服务
└── README.md
```

## 核心功能（MVP）

- [ ] AI角色聊天（文字）
- [ ] AI语音对话
- [ ] 角色浏览和选择
- [ ] 用户注册/登录
- [ ] 基础UI（支持RTL阿拉伯语）

## 技术栈（待定）

- **移动端**: React Native / Flutter
- **AI/LLM**: Google Gemini（阿拉伯语最优）
- **语音**: ElevenLabs TTS + Whisper ASR
- **云平台**: DigitalOcean
- **后端**: TBD

## 对标产品

- [Linky AI](https://play.google.com/store/apps/details?id=com.aigc.ushow.ichat) - AI Chat, Play, Connect
